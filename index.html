症状別トラブルシューティング

症状を選んでピンポイントで解決

「圏外」表示のまま繋がらない場合

  1. eSIMのアクティベーションが完了しているか確認
    • アプリや購入メールで状態をチェック
    • 「有効開始日」が今日かどうか確認
  2. APN設定が正しいか確認（一部のeSIMで必要）
  3. 現地のネットワークを手動選択
    • 設定 → モバイル通信 → ネットワーク選択 → 自動をOFF
    • 表示されるキャリアから選択
  4. eSIMプロバイダーのサポートに問い合わせ
💡 ヒント: 到着後すぐに圏外の場合、空港のWi-Fiに繋いでeSIMアプリを開いて状態確認しましょう。

アンテナは立つがデータ通信できない場合

  1. 「データローミング」の設定を確認（プランの案内に従う）
    • 多くの海外eSIMはローミングONが必要です
    • iPhone: 設定 → モバイル通信 → 該当eSIM → データローミング
    • Android: 設定 → ネットワーク → SIM → ローミング
  2. モバイルデータ通信で使う回線をeSIMに設定
  3. 機内モードON/OFFで再接続
  4. VPNをOFFにする（VPNが原因のこともある）
⚠️ 重要: 日本のSIM（ドコモ・au・ソフトバンク等）のローミングは必ずOFFにしてください！ONのままだと高額請求の原因になります。海外eSIM側のみ設定を変更しましょう。

通信はできるが遅すぎる場合

  1. データ残量を確認（使い切っていないか）
    • 容量超過で速度制限がかかっている可能性
  2. 場所を変えてみる（電波状況が悪い可能性）
  3. 時間帯を変える（混雑時間帯は遅くなる）
  4. ネットワーク設定をリセット
    • iPhone: 設定 → 一般 → リセット → ネットワーク設定
    • Android: 設定 → システム → リセット → ネットワーク設定
  5. データを追加購入、または別のeSIMに切り替え
💡 ヒント: 観光地や繁華街は回線が混み合います。写真のアップロードはホテルのWi-Fiで行うと快適です。

家族旅行｜テザリング vs 各自eSIM

どちらを選ぶ？メリット・デメリット比較

🔗

1台テザリング

✅ メリット

  • 費用を抑えられる（1回線分でOK）
  • 管理が楽（設定は親だけ）
  • データ使用量を把握しやすい

⚠️ デメリット

  • 親機のバッテリー消耗が激しい
  • 親と離れると通信不可
  • 複数台接続で速度低下
向いている人: 常に家族一緒に行動する短期旅行、子供が小さい家族
👨‍👩‍👧

各自eSIM

✅ メリット

  • 別行動でも通信できる
  • 親機のバッテリーを気にしなくてOK
  • 各自が快適な速度で通信

⚠️ デメリット

  • 費用が人数分かかる
  • 各端末の設定が必要
  • 子供端末のeSIM対応確認が必要
向いている人: 別行動あり、長期滞在、子供が中高生以上

よくある質問

海外eSIMに関する疑問をすべて解決

eSIM（Embedded SIM）は端末に内蔵された書き換え可能なSIMです。物理的なSIMカードの抜き差しが不要で、QRコードを読み取るだけで回線を追加できます。海外旅行では、日本のSIMを入れたまま海外の回線を追加できるため、帰国後もすぐに日本の回線に戻せます。

iPhone: 設定 → 一般 → 情報 → 「EID」の項目があればeSIM対応です。iPhone XS以降のモデルが対応しています。

Android: 設定 → ネットワークとインターネット → SIM → 「SIMをダウンロード」や「eSIMを追加」のメニューがあれば対応しています。Pixel 4以降、Galaxy S20以降などが対応しています。

出発の1〜2日前に、自宅のWi-Fi環境でインストールすることをおすすめします。インストール後は「回線をオフ」にしておき、現地到着後にオンにすれば使えます。なお、多くのeSIMは購入後365日以内のアクティベーション有効期限があります。

日本の携帯キャリア（ドコモ・au・ソフトバンク等）のSIMのローミングは必ずOFFにしてください。海外eSIMのローミングをONにしても、購入したデータプラン内で使用されるため追加料金は発生しません。ただし、データを使い切ると追加購入が必要になる場合があります。

多くの海外eSIMは「データ通信専用」です。通話にはLINE通話、FaceTime、WhatsAppなどのインターネット通話アプリを使用してください。一部のeSIMには通話付きプランもありますが、日本人向けにはデータ専用＋通話アプリの組み合わせがコスパ良くおすすめです。

サービスによって異なります。Holaflyはテザリング対応を明記しています。Airaloは一部プランで対応。利用前に各サービスの説明をご確認ください。テザリングを使う場合は、親機のバッテリー消耗が激しくなるためモバイルバッテリー必携です。

お子さんの端末がeSIM対応であれば可能です。ただし、キッズスマホや格安スマホの一部はeSIM非対応のものがあります。また、iPhoneの場合は保護者のApple IDでファミリー共有を使用していても、eSIMの追加は各端末で行う必要があります。

サービスによって対応が異なります。①速度制限がかかる、②完全に通信停止、③自動でデータ追加（有料）のパターンがあります。多くのサービスではアプリから追加データを購入できます。渡航前に余裕を持った容量を選ぶか、無制限プランを選ぶと安心です。

使い切ったeSIMは削除してOKです。設定 → モバイル通信 → 該当eSIM → 「eSIMを削除」で削除できます。削除せずに残しておくこともできますが、古いプランが溜まると管理しづらくなるため、使い終わったら削除することをおすすめします。

はい、ほとんどの海外eSIMサービスは日本発行のVisa/Mastercard/JCBクレジットカードに対応しています。一部のサービスはPayPalやApple Payも使えます。決済画面は英語のことが多いですが、金額とカード情報を入力するだけなので難しくありません。

サービスによってポリシーが異なります。Holaflyは条件付きで返金対応あり。Airaloは未使用・未インストールの場合に返金対応。購入前に各サービスの返金ポリシーをご確認ください。なお、インストール済み・使用開始後の返金は基本的にできません。
